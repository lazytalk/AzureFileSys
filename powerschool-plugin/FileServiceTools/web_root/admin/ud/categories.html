User Data Categories

Categories define "types" of user data (MAP Scores, Club Experience, Skillsets). You can only delete (or edit fields) when no user records exist for that category.

+ New Category

~[tlist_sql; SELECT c.ID cat_id, c.NAME cat_name, c.CODE cat_code, c.IS_ACTIVE cat_active, c.SORT_ORDER cat_sort, (SELECT COUNT(*) FROM U_UD_RECORD r WHERE r.CATEGORY_ID = c.ID) cat_rec_count FROM U_UD_CATEGORY c ORDER BY NVL(c.SORT_ORDER, 9999), UPPER(c.NAME) ] ~[/tlist_sql]
